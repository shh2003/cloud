# 모의투자 사이트

## 프로젝트 구조
```
investment-site/
├── backend/           # Node.js Express 백엔드
│   ├── server.js
│   ├── config/
│   ├── routes/
│   ├── controllers/
│   └── models/
├── frontend/          # React 프론트엔드
│   ├── public/
│   └── src/
└── database/          # DB 스키마 및 초기 데이터
```

### 백엔드
```bash
cd backend
npm install
npm start
```

### 프론트엔드
```bash
cd frontend
npm install
npm start
```
